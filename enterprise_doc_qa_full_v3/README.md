# Enterprise Document Q&A - Full v3 Bundle

This bundle includes sample PDFs and DOCX documents located in `data/sample_docs`.

Run locally:
1. Create venv and install: pip install -r requirements.txt
2. streamlit run src/app.py
3. Click 'Index documents' then ask questions.

Add your OpenAI key: copy config/keys_template.py -> config/keys.py and set OPENAI_API_KEY.
